BuySmart es una aplicación de escritorio desarrollada en Python con Tkinter que funciona como un gestor de listas de supermercado y control de gastos personales.
Su objetivo es facilitar la organización de compras y el control del presupuesto mensual de cada usuario.
-Características principales
Gestión de listas de compras: crea, agrega y elimina listas de supermercado.
Control de gastos personales: registra tus compras y controla tu presupuesto mensual.
Resumen del mes: muestra un balance de tus gastos.
Interfaz simple e intuitiva: creada con Tkinter, ideal para uso cotidiano.
-Tecnologías utilizadas
Lenguaje: Python 3
Interfaz gráfica: Tkinter
Gestión de proyecto: Trello y GitHub
